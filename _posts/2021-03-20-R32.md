<h1 id="capital--budgeting">Capital  Budgeting</h1>
<h2 id="the--basic--of-capital--budgeting">The  Basic  of Capital  Budgeting</h2>
<blockquote>
<p>Capital  budgeting  is  the  process  of  determining  and  selecting  the  most profitable  long-term  projects.</p>
</blockquote>
<h3 id="钱从哪来？">1. 钱从哪来？</h3>
<p>Liability （Debt）e.g. 40%<br>
Equity		 (Stock)   e.g. 60%<br>
Liability + Equity = Cost of Capital (融资成本)<br>
(债成本比股小)</p>
<p><a href="https://www.codecogs.com/eqnedit.php?latex=r_d&amp;space;=&amp;space;8%,&amp;space;~~r_e=12%" target="_blank"><img src="https://latex.codecogs.com/gif.latex?r_d&amp;space;=&amp;space;8%,&amp;space;~~r_e=12%" title="r_d = 8%, ~~r_e=12%"></a></p>
<p><em><strong>WACC: Weighted Average Cost of Capital</strong></em><br>
e.g.  <a href="https://www.codecogs.com/eqnedit.php?latex=WACC&amp;space;=&amp;space;W_d\times&amp;space;r_d&amp;space;+&amp;space;W_e&amp;space;\times&amp;space;r_e" target="_blank"><img src="https://latex.codecogs.com/gif.latex?WACC&amp;space;=&amp;space;W_d\times&amp;space;r_d&amp;space;+&amp;space;W_e&amp;space;\times&amp;space;r_e" title="WACC = W_d\times r_d + W_e \times r_e"></a></p>
<h3 id="钱往哪花？">2. 钱往哪花？</h3>
<p>投资</p>
<ul>
<li>长期： 1年以上，Capital budgeting （资本预算，五个指标）</li>
<li>短期：
<ul>
<li>Inventory: Current Assets 流动资产</li>
<li>Payable (应付): Current Liability 流动负债</li>
</ul>
</li>
</ul>
<p>Current Assets - Current Liabilities = <em><strong>Working Capital</strong></em> (WC 营运资本)</p>
<p><em><strong>营运资本管理</strong></em></p>
<ul>
<li>Cash</li>
<li>Inventory</li>
<li>A/R (Account Receivable)</li>
<li>A/P (Account Payable)</li>
</ul>
<h3 id="三类杠杆-leverage">3. 三类杠杆 Leverage</h3>
<ul>
<li>ROE （Return on Equity）<br>
<a href="https://www.codecogs.com/eqnedit.php?latex=ROE&amp;space;=&amp;space;\frac{Net~Income}{Equity}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?ROE&amp;space;=&amp;space;\frac{Net~Income}{Equity}" title="ROE = \frac{Net~Income}{Equity}"></a><br>
Equity: 平均所有者权益</li>
<li>ROA (Return on Asset)<br>
<a href="https://www.codecogs.com/eqnedit.php?latex=ROA&amp;space;=&amp;space;\frac{Net~Income}{Asset}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?ROA&amp;space;=&amp;space;\frac{Net~Income}{Asset}" title="ROA = \frac{Net~Income}{Asset}"></a><br>
Asset: 平均总资产 = Liability + Equity<br>
<a href="https://www.codecogs.com/eqnedit.php?latex=ROE&amp;space;=&amp;space;ROA&amp;space;\times&amp;space;\frac{Asset}{Equity}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?ROE&amp;space;=&amp;space;ROA&amp;space;\times&amp;space;\frac{Asset}{Equity}" title="ROE = ROA \times \frac{Asset}{Equity}"></a><br>
A/E is the financial Leverage</li>
</ul>
<p><em><strong>增加A/E 可以提高股东收益</strong></em><br>
<em><strong>Debt借债增加，股东回报率越高</strong></em></p>
<p>Trade off:</p>
<ul>
<li>Debt 增加 -&gt; leverage增加 -&gt; 放大收益</li>
<li>Debt 增加 -&gt; credit risk 增加 -&gt; 财务风险增加</li>
</ul>

<h2 id="capital-budgeting-资本预算">Capital Budgeting 资本预算</h2>
<blockquote>
<p>选择最赚钱的长期项目进行投资的过程</p>
</blockquote>
<p>资本预算过程</p>
<ol>
<li>Idea generation 想法产生</li>
<li>Analysing project proposals 分析，估值 （分析被收购方）</li>
<li>Create the firm-wide capital budget 分析自己公司
<ul>
<li>是否符合公司长期战略</li>
<li>现金流时间点</li>
<li>资源是否满足</li>
</ul>
</li>
<li>Monitoring decisions and conducting a post-audit 投后管理
<ul>
<li>分析投资前后，差异原因</li>
<li>Comparing  the  actual  result  with  the  projected  and  explain  the  reasons</li>
</ul>
</li>
</ol>
<p><em><strong>项目分类</strong></em></p>
<ul>
<li>Replacement project 新项目代替老项目
<ul>
<li>maintain the business</li>
<li>cost reduction (替代项目主要是为了降低成本)</li>
</ul>
</li>
<li>Expansion project 扩张性项目
<ul>
<li>扩大产量
<ul>
<li>对现有产品线扩张</li>
<li>新项目，新产品扩张 （高风险）</li>
</ul>
</li>
</ul>
</li>
<li>Mandatory investment 强制性项目
<ul>
<li>e.g. Regulatory, safe, and environmental project</li>
<li>不是为了经济利润</li>
</ul>
</li>
<li>other project
<ul>
<li>现金流很难预估的项目</li>
<li>e.g. 请明星代言</li>
</ul>
</li>
</ul>
<h2 id="basic-principles-of-capital-budgeting-资本预算原则（5个）">Basic principles of capital budgeting 资本预算原则（5个）</h2>
<ol>
<li><strong>Decisions are based on incremental cash flow，not  accounting  income</strong> 为了增加现金流</li>
</ol>
<ul>
<li>e.g. 4台自动机器（产生100w现金流）代替4台手动机器（产生80万现金流）</li>
<li>incremental CF： 100w - 80w = 20w</li>
</ul>
<ol start="2">
<li><strong>The timing of cash flow is important</strong></li>
</ol>
<ul>
<li>time value of money</li>
<li>现金流时间点一致才可以比较</li>
<li>Cash  flows  received  earlier  are  worth  more  than  cash  flows  to  be received:  accelerated  depreciation.</li>
</ul>
<ol start="3">
<li><strong>Cash flow are analyzed on an after-tax basis</strong>  税后比较</li>
<li><strong>Cash flow should be included：</strong></li>
</ol>
<ul>
<li>Externalities  外部性 -&gt; 一个项目对外部环境的影响 -&gt; 项目对公司其他项目的影响
<ul>
<li>negative: cannibalization 吞噬效应 （新项目吞噬老项目的市场份额）</li>
<li>positive: 新项目对老项目有促进作用</li>
</ul>
</li>
<li>Opportunity cost 机会成本</li>
</ul>
<ol start="5">
<li><strong>Cash flow should be ignored：</strong></li>
</ol>
<ul>
<li>sunk costs 沉默成本
<ul>
<li>与未来现金流无关的成本（付出后无法收回的成本）</li>
<li>e.g. 咨询费用</li>
</ul>
</li>
<li>Financing costs/interest costs<br>
<a href="https://www.codecogs.com/eqnedit.php?latex=NPV=\sum^{n}_{t=0}\frac{CF_t}{(1+r)^t}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?NPV=\sum^{n}_{t=0}\frac{CF_t}{(1+r)^t}" title="NPV=\sum^{n}_{t=0}\frac{CF_t}{(1+r)^t}"></a>
<ul>
<li>NPV: 净现值</li>
<li>r：融资成本
<ul>
<li>债 r_d = 8%</li>
<li>股 r_e= 12%</li>
<li>可以根据WACC算得均值</li>
</ul>
</li>
<li><a href="https://www.codecogs.com/eqnedit.php?latex=CF_t&amp;space;=&amp;space;CF_t^{in}&amp;space;-&amp;space;CF_t^{out}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?CF_t&amp;space;=&amp;space;CF_t^{in}&amp;space;-&amp;space;CF_t^{out}" title="CF_t = CF_t^{in} - CF_t^{out}"></a></li>
</ul>
</li>
</ul>
<p><strong>Independent projects</strong> 独立项目 项目间不互相影响，决策可以独立<br>
<strong>Mutually exclusive projects</strong> 互斥项目 （e.g. 资金有限，投A就不能投B）</p>
<p><strong>project sequencing</strong> 投资是有顺序的</p>
<p>公司资金情况</p>
<ul>
<li>unlimited funds -&gt;所有项目都相当于是独立项目</li>
<li>Capital rationing 定量配给</li>
</ul>
<h2 id="project-evaluation-methods-5个指标">Project Evaluation Methods (5个指标)</h2>
<ol>
<li>Net Present Value (NPV) 净现值</li>
<li>Internal Rate of Return (IRR) 内部回报率</li>
<li>Payback Period (PBP)  回收期</li>
<li>Discount Payback Period (DPBP) 折现回收期</li>
<li>Profitability Index (PI) 盈利指数</li>
</ol>
<h3 id="net-present-value-npv">Net Present Value (NPV)</h3>
<p><a href="https://www.codecogs.com/eqnedit.php?latex=NPV=\sum^{n}_{t=0}\frac{CF_t}{(1+r)^t}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?NPV=\sum^{n}_{t=0}\frac{CF_t}{(1+r)^t}" title="NPV=\sum^{n}_{t=0}\frac{CF_t}{(1+r)^t}"></a><br>
<a href="https://www.codecogs.com/eqnedit.php?latex=CF_t&amp;space;=&amp;space;CF_t^{in}&amp;space;-&amp;space;CF_t^{out}" target="_blank"><img src="https://latex.codecogs.com/gif.latex?CF_t&amp;space;=&amp;space;CF_t^{in}&amp;space;-&amp;space;CF_t^{out}" title="CF_t = CF_t^{in} - CF_t^{out}"></a><br>
cost of capital 就是算好的折现率<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goqb9hmgegj30bb01vjr8.jpg" alt="image.png"><br>
<strong>Decision Rule</strong></p>
<ul>
<li>For independent projects
<ul>
<li>if NPV &gt; 0 可投</li>
<li>if NPV &lt;0 不投</li>
</ul>
</li>
<li>For mutually exclusive project
<ul>
<li>投NPV最大的项目</li>
</ul>
</li>
</ul>
<p><strong>NPV advantages</strong></p>
<ul>
<li>利润按绝对金额计算 Shows  the  amount  of  gains  as  currency  amount</li>
<li>反映项目给股东及公司的增值</li>
<li>现实折价率 Realistic  discount  rate  –  included  opportunity  cost  of  funds  (the expected  return  of  stockholders). 考虑了机会成本及现实成本， 与IRR相比</li>
</ul>
<p><strong>NPV disadvantages</strong><br>
没有考虑初始投资规模，只看了绝对利润</p>
<h3 id="irr：-internal-rate-of-return-内部回报率">IRR： Internal rate of return 内部回报率</h3>
<p>本质就是NPV=0的折现率 ，NPV等于0时候的cost of capital<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goqbonbbflj30hu01mwed.jpg" alt="image.png"><br>
相当于考虑了投资规模 -&gt; 收益率<br>
<strong>Decision rule</strong></p>
<ul>
<li>For independent projects
<ul>
<li>if IRR&gt; required rate of return, 可投</li>
<li>if NPV &lt; required rate of return, 不投</li>
</ul>
</li>
<li>For mutually exclusive project
<ul>
<li>投IRR最大的项目<br>
<strong>IRR advantages</strong></li>
</ul>
</li>
<li>可以用来比较不同规模的投资</li>
<li>反映profitability</li>
<li>不是绝对金额，而是相对金额</li>
</ul>
<p><strong>IRR disadvantages</strong></p>
<ul>
<li>假设再投资收益率就是IRR，（unrealistic）</li>
<li>解方程可能无解，或者多解</li>
</ul>
<blockquote>
<p>当IRR与NPV有冲突时，优先按NPV选择</p>
</blockquote>
<p><img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goqbvo7fh8j30pp0fggn2.jpg" alt="image.png"></p>
<blockquote>
<p>CF符号变动几次就可能最多有几个IRR<br>
横纵坐标交点含义！<br>
(看笔记！！)</p>
</blockquote>

<h3 id="pi-profitability-index-盈利指数">PI: Profitability Index (盈利指数)</h3>
<p>The  PV  of  a  project’s  future  cash  flows  divided  by  the  initial investment<br>

<h2 id="impact--of--npv--rule--and--stock--price">Impact  of  NPV  Rule  and  Stock  Price</h2>
<p>NPV如何影响股票价格</p>
<ol>
<li>定性</li>
</ol>
<ul>
<li>NPV &gt; 0 , 公司赚钱， 股票上涨</li>
<li>NPV = 0， 公司不赚不亏， 股票不变</li>
<li>NPV &lt; 0, 公司亏钱， 股票下跌</li>
</ul>
<ol start="2">
<li>定量<br>
新的股票价格<br>
P = NPV/总股数+股票现价</li>
<li>现实生活中，如果预期的NPV比实际的产生的NPV大，也可能会造成股票下跌，因为预期落差。</li>
</ol>
