<p>By Ray Dalio</p>
<h2 id="三股经济动力">三股经济动力</h2>
<ol>
<li>生产率提高</li>
<li>短期债务周期</li>
<li>长期债务周期<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goukti4ghwj30lu05h449.jpg" alt="image.png"></li>
</ol>
<p>购买物品可以用货币或者信用<br>
支出总额是经济的驱动力<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goukv6l4wgj30jt07tgty.jpg" alt="image.png"></p>
<p>价格 = 支出金额/销量<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goukwzbq0mj30l70an4a8.jpg" alt="image.png"></p>
<p>用货币和信用交换商品，服务和金融资产</p>
<p>政府有两个组成部分</p>
<ul>
<li>中央政府：收税</li>
<li>中央银行：花钱
<ul>
<li>控制经济中的货币和信贷数量</li>
<li>通过影响利率和发行更多货币来控制</li>
</ul>
</li>
</ul>
<h2 id="信贷">信贷</h2>
<p>经济中最重要的组成部分<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goul20xpbhj30nx0gp1bk.jpg" alt="image.png"></p>
<p>信贷一旦产生就成为债务<br>
债务是贷款人的资产，是借款人的负债</p>
<p>信用良好的借款人有两个条件：</p>
<ul>
<li>偿还能力</li>
<li>不能还债时的抵押物</li>
</ul>
<p>生产率在长期最关键，而信贷在短期最重要</p>
<ul>
<li>这是因为生产率的提高不会剧烈波动</li>
<li>但债务可以让消费时超出产出，而在还债时消费低于产出</li>
</ul>
<p>债务率的波动有两大周期<br>
一个周期大概持续5-8年<br>
一个持续75-100年</p>
<p>如果没有信贷，则经济发展与生产率提升是线性关系，而因为有了信贷，则形成了周期。</p>
<p>美国国内信贷总额大概50万亿美圆，而货币总额大概只有3万亿<br>
<img src="http://ww1.sinaimg.cn/large/0072j28Pgy1goulcp77jxj30sy0g6wzv.jpg" alt="image.png"></p>
<h3 id="短期债务周期">短期债务周期</h3>
<ul>
<li>支出增加，价格开始上涨，</li>
<li>支出增加的是信贷，而信贷是凭空产生的。支出和收入的增长速度超过出售商品的生产速度，供小于求，价格上涨 -&gt; 通货膨胀-&gt;经济活动加速。</li>
<li>央行不希望价格上涨，则会提高利率，利率上涨，借贷成本上升，借贷减少，还款额增加，用于支出的资金减少。支出速度放慢，价格下跌 -&gt;通货紧缩 -&gt;经济开始衰退 -&gt;央行开始降低利率。。。</li>
<li>信贷易于获得，经济就会扩张，信贷不易获得，经济就会减速，这个周期由央行控制</li>
</ul>
<h3 id="长期债务周期">长期债务周期</h3>
<p>经济欣欣向荣时，人们会过度借债消费，形成泡沫<br>
债务与收入的比率称为<strong>债务负担</strong><br>
当经济下行时，债务会变的难以负担，国家进入<strong>去杠杆化</strong>。长期债务周期下行阶段，因为人们收入急剧减少，无法通过降低利率来挽回局面。例如美国在2008年去杠杆化阶段，利率就是降到0。</p>
<p>如何应对去杠杆化？<br>
问题在于债务负担过重，必须减轻，四种办法</p>
<ol>
<li>消减支出 （个人，企业和政府）
<ul>
<li>紧缩</li>
<li>支出紧缩往往带来收入紧缩，进入通货紧缩</li>
</ul>
</li>
<li>减少债务 （通过债务违约和重组）
<ul>
<li>债务违约给银行造成坏账，人们因担心银行信用而去银行挤兑，这种严重的经济收缩就是<strong>萧条</strong></li>
<li>债务重组以为贷款人得到的还款减少或者还期延长，或利率低于协定水平。</li>
</ul>
</li>
<li>财富再分配 （将部分财富从富人转给穷人）
<ul>
<li>收入降低，失业率上升，税收减少而补助支出上升。政府预算赤字飙升</li>
<li>政府资金来源就变成的富人 -&gt; 加税</li>
<li>造成社会对立</li>
</ul>
</li>
<li>发行货币
<ul>
<li>央行不得不发行更多货币来购买金融资产和政府债券。通过大量购买来提高资产价格，从而提高了人们的信用。但这仅仅帮助了有金融资产的人。</li>
<li>而政府可以购买商品和服务，向人民送钱，却无法印钞票</li>
<li>因此为了刺激经济，央行和政府就必须合作。央行通过购买政府债券，把钱借给政府，偿还赤字。政府通过刺激计划和失业救济金来增加人们收入，<strong>也增加了政府的债务</strong>。但这个方法降低了经济中的总债务负担。</li>
<li>政府有很大风险，平衡通货膨胀与通货紧缩，适当的平衡就和谐的去杠杆化。</li>
</ul>
</li>
</ol>
<p>如果增发的货币抵消了信贷的降幅，就不会通货膨胀。因为支出总额 = 信贷 + 货币，消失的信贷可以通过发行货币弥补。而价格等于支出总额/产销量，支出总额不变导致价格不波动。</p>
<p>通过发行货币，需要让收入的增长率超过所积累的债务利率（收入一定要比债务增长的快），这样债务负担才可能逐步减轻。</p>
<h3 id="三条经验法则">三条经验法则</h3>
<ol>
<li>不要让债务的增长速度超过收入。</li>
<li>不要让收入的增长速度超过生产率。</li>
<li>尽一切努力提高生产率。</li>
</ol>
